# MX Cloud PDV

Sistema PDV Multi-segmento desenvolvido em Flutter com design adaptativo.

## Características

- ✅ Layout adaptativo para Mobile, Tablet e Desktop
- ✅ Interface baseada em cards
- ✅ Suporte a múltiplos segmentos (Restaurante, Oficina, Varejo, Serviços)
- ✅ Design moderno e intuitivo
- ✅ Tema customizado

## Estrutura do Projeto

```
lib/
├── core/
│   ├── adaptive_layout/    # Sistema de layout adaptativo
│   └── theme/              # Tema e estilos
├── screens/
│   └── home/               # Tela inicial
└── widgets/                # Componentes reutilizáveis
```

## Como executar

```bash
# Instalar dependências
flutter pub get

# Executar em modo debug
flutter run

# Executar em modo release
flutter run --release
```
